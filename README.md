# Poison

Posion: is a script capable of hosts scanning, spoofing MAC, SYNflood attack and FTP sniffer.

![poison](https://user-images.githubusercontent.com/75953873/154956561-6a545954-b6e7-4f47-849b-f16a5dc64994.png)


### Scan Hosts commands
| ARGUMENT | DESCRIPTION |
| ------------- | ------------- |
| -m / --menu | Option menu  |
| -r / --range  | Scan hosts  |


### Spoofing commands
| ARGUMENT | DESCRIPTION |
| ------------- | ------------- |
| -t / --target | Target to spoof  |
| -s / --spoofed  | IP router  |


### Synflood commands
| ARGUMENT | DESCRIPTION |
| ------------- | ------------- |
| -syn / --synflood | Synflood Attack  |


### FTP Sniff commands
| ARGUMENT | DESCRIPTION |
| ------------- | ------------- |
| -i / --interface | FTP sniff  |

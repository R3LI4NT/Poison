# Poison

Posion: is a script capable of hosts scanning, spoofing MAC, SYNflood attack and sniffing FTP.

### Scan Hosts
| ARGUMENT | DESCRIPTION |
| ------------- | ------------- |
| -m / --menu | Option menu  |
| -r / --range  | Scan hosts  |


### Spoofing
| ARGUMENT | DESCRIPTION |
| ------------- | ------------- |
| -t / --target | Target to spoof  |
| -s / --spoofed  | IP router  |


### Synflood
| ARGUMENT | DESCRIPTION |
| ------------- | ------------- |
| -syn / --synflood | Synflood Attack  |


### FTP Sniff
| ARGUMENT | DESCRIPTION |
| ------------- | ------------- |
| -i / --interface | FTP sniff  |
